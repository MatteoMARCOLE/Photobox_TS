Projet Photobox TS

Membre du groupe :
- Mattéo Marcolé

Fonctionnalités réalisées :
- Chargement d'une photo depuis l'API Photobox
- Affichage d'une photo dans la page avec ses informations
- Récupération de l'identifiant de la photo depuis l'URL avec le hash
- Affichage de la catégorie de la photo
- Affichage des commentaires de la photo
- Affichage d'une galerie de photos
- Navigation dans la galerie avec les boutons première, précédente, suivante et dernière page
- Affichage d'une photo au clic sur une vignette
- Affichage optionnel d'une photo en lightbox

Pour lancer le projet :

1. Cloner le dépôt

git clone https://github.com/MatteoMARCOLE/Photobox_TS.git

2. Se placer dans le dossier du projet

cd Photobox_TS

3. Installer les dépendances

npm install

4. Compiler le projet

npm run build

5. Ouvrir le fichier index.html dans un navigateur

Remarque :
Le dossier node_modules n'est pas présent dans le dépôt.
Il est recréé avec la commande npm install.

Accès à l'API :
Si vous n'êtes pas connecté au réseau de l'IUT, il peut être nécessaire d'utiliser le VPN de l'université pour accéder à l'API Photobox.
