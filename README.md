Projet Photobox TS

Pour lancer le projet :

1. Cloner le dépôt

git clone https://github.com/MatteoMARCOLE/Photobox_TS.git

2. Se placer dans le dossier du projet

cd Photobox_TS

3. Installer les dépendances

npm install

4. Compiler le projet

npm run build

5. Ouvrir le fichier index.html dans un navigateur

Remarque :
Le dossier node_modules n'est pas présent dans le dépôt.
Il est recréé avec la commande npm install.
